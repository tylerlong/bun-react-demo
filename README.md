## Instructions

### Compile TS and assets

```
bun w
```

### Compile scss

```
bun css
```

### Start a HTTP Server

```
bun s
```

### Check the website

http://localhost:3000

## Notes

In your \*.scss files, if you want to include an extension css file:

```scss
@use "../node_modules/antd/dist/reset";
```
